# 🚗 Car Rental Management System

A web-based Car Rental Management System developed using PHP and MySQL.  
This system allows users to register, login, browse available cars, book vehicles, and manage rentals efficiently.  
It also includes an admin dashboard for managing cars, users, and bookings.

---

## 📌 Project Overview

The Car Rental Management System is designed to automate the process of renting cars online.  
It reduces manual paperwork and provides a user-friendly interface for both customers and administrators.

---

## ✨ Features

### 👤 User Features
- User Registration & Login
- Browse Available Cars
- View Car Details
- Book a Car
- View Booking History
- Profile Management

### 🛠 Admin Features
- Admin Login Dashboard
- Add / Edit / Delete Cars
- Manage Users
- Manage Bookings
- View Rental Reports

## ⚙️ Installation Guide

Follow these steps to run the project locally:

### 1️⃣ Install XAMPP
Download and install XAMPP:
https://www.apachefriends.org/index.html

### 2️⃣ Move Project Folder
Copy the project folder into:

C:\xampp\htdocs\


### 3️⃣ Start Server
Open XAMPP Control Panel and start:
- Apache
- MySQL

### 4️⃣ Create Database
1. Open browser and go to:

http://localhost/phpmyadmin

2. Create new database named:

car_rental

3. Import the file:

car_rental.sql


### 5️⃣ Run the Project
Open browser:

http://localhost/car-rental
